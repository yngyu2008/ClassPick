# ClassPick

학급 자리배치를 쉽고 빠르게 할 수 있는 자리배치 프로그램입니다.  
학년과 반별로 학생 명단과 자리배치 상태를 저장할 수 있으며, 자리 고정, 자리 교환, 랜덤 자리배치, 자리표 이미지 내보내기 기능을 지원합니다.  
또한 5열 / 6열 배열 설정과 창 크기 변화에 대응하는 화면 구성을 통해 다양한 교실 환경에 맞게 사용할 수 있습니다.

## 1. 프로젝트 소개

ClassPick은 학교에서 사용할 자리배치표를 편리하게 만들기 위해 제작한 프로그램입니다.  
학생 명단을 입력한 뒤 자동으로 자리를 배치할 수 있고, 특정 자리를 고정하거나 직접 자리를 바꾸는 것도 가능합니다.  
또한 마지막으로 작업하던 학급 정보를 기억하여, 프로그램을 다시 실행했을 때 이전에 보던 자리배치 상태를 바로 불러올 수 있습니다.

## 2. 개발 환경

- Language: Python
- Editor: Visual Studio Code
- GUI: Tkinter
- Database: SQLite

## 3. 주요 기능

### 3-1. 학년 / 반별 데이터 저장
- 학년과 반은 각각 독립된 데이터로 저장됩니다.
- 각 학급별로 학생 명단, 자리배치 상태, 배열 설정이 함께 저장됩니다.
- 예를 들어 3학년 7반을 저장한 뒤 다시 불러오면 이전 자리배치 상태가 그대로 복원됩니다.
- 프로그램 종료 후 다시 실행하면 마지막으로 작업하던 학급의 자리배치표가 자동으로 열립니다.

### 3-2. 학생 명단 입력
- 학생 이름은 한 줄에 한 명씩 입력합니다.
- 번호를 직접 입력하지 않아도 자동으로 `01. 이름` 형식으로 정리됩니다.
- 입력된 순서를 기준으로 초기 자리표에 이름이 표시됩니다.

### 예시
01. 김철수  
02. 신짱구  
03. 이훈이  
04. 한유리

### 3-3. 배열 설정
- 설정 창에서 5열 또는 6열 배열을 선택할 수 있습니다.
- 5열 배열은 5행 5열 구조로, 최대 25명까지 배치할 수 있습니다.
- 6열 배열은 5행 6열 구조로, 최대 30명까지 배치할 수 있습니다.
- 학생 수가 최대 좌석 수보다 적으면 맨 뒤 오른쪽 자리부터 차례대로 제거됩니다.

### 3-4. 짝 설정
- 짝 있음 : 2칸씩 붙어 있는 형태로 자리표가 표시됩니다.
- 짝 없음 : 모든 자리가 일정 간격으로 떨어져 표시됩니다.
- 짝 설정은 6열 배열에서만 사용할 수 있습니다.
- 5열 배열을 선택하면 짝 설정은 비활성화되며 자동으로 짝 없음이 적용됩니다.

### 3-5. 자리 선택 및 교환
- 고정되지 않은 자리를 클릭하면 해당 자리가 선택됩니다.
- 선택된 자리는 붉은색 테두리로 표시됩니다.
- 이후 다른 고정되지 않은 자리를 클릭하면 두 자리의 학생 이름이 서로 바뀝니다.
- 같은 자리를 다시 클릭하면 선택이 해제됩니다.

### 3-6. 자리 고정
- 각 자리 오른쪽 아래의 자물쇠 아이콘으로 고정 여부를 변경할 수 있습니다.
- 고정된 자리는 랜덤 자리배치 대상에서 제외됩니다.
- 고정된 자리는 수동 교환도 불가능합니다.

### 3-7. 랜덤 자리배치
- 자리배치 버튼을 누르면 고정되지 않은 자리만 랜덤으로 섞입니다.
- 고정된 자리는 그대로 유지됩니다.
- 자리배치 중에는 이름이 바뀌는 애니메이션이 표시됩니다.

### 3-8. 자리표 이미지 내보내기
- 내보내기 버튼을 누르면 현재 자리배치표가 PNG 파일로 저장됩니다.
- 저장 위치는 바탕화면입니다.
- 내보내기 시 현재 설정이 자동으로 저장됩니다.
- 파일명에는 저장 날짜와 학년 / 반 정보가 포함됩니다.

### 3-9. 창 크기 대응 및 전체화면 사용
- 메인 화면은 창 크기에 맞춰 좌석 위치를 다시 계산합니다.
- 창을 최대화하거나 크기를 변경해도 좌석 UI가 최대한 자연스럽게 유지되도록 구성되어 있습니다.
- 전체화면 전환을 지원하며, 큰 화면에서도 자리배치를 편하게 확인할 수 있습니다.

## 4. 기본 사용 방법
1. 프로그램을 실행합니다.
2. 상단의 설정 버튼을 클릭합니다.
3. 학년과 반을 입력합니다.
4. 학생 이름을 입력합니다.
5. 배열 설정에서 5열 또는 6열을 선택합니다.
6. 6열인 경우 필요에 따라 짝 있음 / 짝 없음을 선택합니다.
7. 적용 버튼을 클릭합니다.
8. 메인 화면에서 자리배치 및 수정을 진행합니다.
9. 필요할 때 저장하거나 내보내기 기능을 사용합니다.

## 5. 좌석 배치 규칙
- 좌석은 기본적으로 5행 구조입니다.
- 5열 배열 선택 시 총 25자리까지 사용할 수 있습니다.
- 6열 배열 선택 시 총 30자리까지 사용할 수 있습니다.
- 학생 수가 좌석 수보다 적으면 맨 뒤 오른쪽 자리부터 차례대로 비활성화됩니다.
- 고정된 자리는 랜덤 자리배치에서 유지됩니다.

## 6. 실행 및 설치

### 필수 사항
- Python이 설치되어 있어야 합니다.

### 라이브러리 설치
- PNG 내보내기 기능을 사용하려면 Pillow 라이브러리가 필요합니다.
- 아래 명령어를 터미널에 입력해 설치할 수 있습니다.

```bash
pip install pillow
```
### 7. 조작 방법
- 설정 : 학년, 반, 학생 명단, 배열, 짝 설정을 변경합니다.
- 저장 : 현재 학급의 자리배치 상태를 저장합니다.
- 내보내기 : 현재 자리배치표를 PNG 파일로 저장합니다.
- 자리 클릭 : 자리 선택 및 자리 교환을 수행합니다.
- 자물쇠 클릭 : 해당 자리의 고정 여부를 변경합니다.
- F11 : 전체화면 전환
- Esc : 전체화면 해제

### 8. 주의사항
- 5열 배열은 최대 25명, 6열 배열은 최대 30명까지 입력할 수 있습니다.
- 5열 배열에서는 짝 있음 기능을 사용할 수 없습니다.
- Pillow가 설치되어 있지 않으면 PNG 내보내기 기능이 정상적으로 동작하지 않을 수 있습니다.
- 저장 파일이 삭제되거나 손상되면 기존 학급 정보와 마지막 작업 상태가 복원되지 않을 수 있습니다.

### 9. 프로그램 용도
이 프로그램은 학급 자리배치표를 빠르고 편리하게 작성하고 관리하기 위한 목적으로 제작되었습니다.
반별 학생 명단 관리, 자리 고정, 자리 교환, 랜덤 배치, 이미지 저장 기능을 통해 실제 학급 운영에 활용할 수 있습니다.
